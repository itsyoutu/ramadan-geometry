# منصة القيم الرمضانية الهندسية

## 🌙 مشروع هندسة القيم الرمضانية

مشروع تفاعلي يجمع بين **الرياضيات** و**الهندسة** و**القيم الإسلامية** لحساب مواقيت الصيام في دولة قطر.

### 👨‍🎓 المصمم
**يوسف وليد محمد**  
الصف 8/1 - مدرسة أبو بكر الصديق الإعدادية للبنين

### ✨ المميزات الرئيسية

1. **محاكاة هندسية تفاعلية** - تتبع حركة الشمس والزوايا الهندسية
2. **فيديو توضيحي** - شرح مفصل لآلية عمل المشروع
3. **عرض تقديمي** - 13 شريحة تغطي جميع جوانب المشروع
4. **معادلات رياضية** - شرح المعادلات المستخدمة في الحسابات

### 🔧 التقنيات المستخدمة

- **HTML5 & CSS3** - تصميم واجهة المستخدم
- **JavaScript** - المنطق التفاعلي والحسابات
- **Canvas API** - رسم المحاكاة الهندسية
- **Tailwind CSS** - تنسيق احترافي
